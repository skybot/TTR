The COMPUTER setup store all information about all you computer in your network.
If you what to add a new Computer to your network got to SETUPComputer
And input the following data:
  • MAC Address
  • PC's MAC Address
  • PC's IP Address
  • PC's Location
  • Computer Type
  • Computer visible:
It is possible to include a new Computer to a already commissioned room! Do so by input the upper values correctly and press the RECONFIGURE SERVER Button.
If you you what to make a new computer inside a new room please have a look at SETUPRoom
The Computer Type is used to give a better overview of your used hardware. If you want to input a already known type go on, if you what to add a new kind of hardware go to SETUPHardware
SETUPComputer can also be used if one of your PC`s is broken and you need to change the Main board. TAC RECOVERY always recognize A Computer by his MAC Address. So if you input a new Network card or Main board you get a new MAC Address.
In this case search inside the TABLE with all your Computer for the Computer witch is broken, and mark this line. Bush the button Edit at the top of the Table, and change the old MAC Address to the new one. If you finished this please press the RECONFIGURE SERVER Button.
